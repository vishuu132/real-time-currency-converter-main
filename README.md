## 📌 Overview

CurrencyConverter is a responsive and user-friendly Currency Converter application built using **HTML, CSS, and JavaScript**.  
The application fetches live exchange rates from an API and provides quick and accurate currency conversion in real time.

This project helped me strengthen my understanding of:
- API Integration
- Asynchronous JavaScript
- DOM Manipulation
- Responsive Web Design

---

# 🚀 Features

✅ Real-time currency conversion  
✅ Live exchange rates using API  
✅ Responsive design for all devices  
✅ Country flag integration 🌍  
✅ Swap currency functionality 🔄  
✅ Clean and modern UI
